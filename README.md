# 部署说明

> 本人线上环境如下

系统 | CPU | 内存
---|---|---
CentOS7.6 | 2核 | 4 GB

- 如果是一台 `1 GB` 内存的服务器，肯定不够跑
- 如果是一台 `2 GB` 内存的服务器，基本能跑
- 如果是一台 `4 GB` 内存的服务器，无压力
